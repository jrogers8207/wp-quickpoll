== Description ==

WordPoll is a plugin for Wordpress that allows users to create polls on their website and see the results.

== Installation ==

1. Install the folder in '/wp-content/plugins/'.
2. Go to your Wordpress dashboard.
3. Click on "Plugins".
4. Activate the plugin "WordPoll".
